## **[R] 470. Implement Rand10() Using Rand7()**

### CODE
```
class Solution:
    def rand10(self):
        """
        :rtype: int
        """
        first = rand7()
        second = rand7()
        
        x = (first - 1) * 7 + second
        
        if x <= 40:
            r = x % 10
            return 10 if r == 0 else r
        else:
            return self.rand10()
```
